# Java Number Analyzer

A simple Java console application that demonstrates core Java concepts.

## Features
- Checks whether a number is even or odd
- Identifies positive, negative, or zero
- Prints multiplication table up to 10

## Concepts Used
- Java Basics
- Methods
- Conditional Statements
- Loops
- Scanner (User Input)

## How to Run
1. Compile the program:
   javac NumberAnalyzer.java
2. Run:
   java NumberAnalyzer

## Sample Output
Enter a number: 5  
5 is Odd  
5 is Positive  
Multiplication Table:
5 x 1 = 5  
...
